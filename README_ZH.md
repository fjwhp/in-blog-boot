# ibo 博客系统


> ibo i:imprint 形容的是印记:印记博客.想必大家都希望自己的博客记录自己所想,留有所念,留下胜过的印记

`ibo` 使用了spring-boot 、 mongodb 

演示站点：https://www.ijson.net

> 本博客开发周期按月计算,毕竟平时都上班,不会天天来编写代码,只能周六日去编写此代码,按月发布

# 使用技术
1. SpringBoot 2.1.6
2. Mongodb 3.8.2
4. ibeetl 2.7.22


# 特性

* 支持Markdown等软文发布
* 设计简洁，界面美观
* 支持文章附件单独存储
* 部署简单，不依赖 Tomcat
* 支持mongodb
